Turn-based RPG developed in Godot Engine with original pixel art. Players explore a world where magic and programming intertwine, solving logic puzzles and facing enemies in grid-based tactical combat.

## Game Features

- Tactical turn-based combat on isometric grid
- 50+ abilities inspired by programming concepts
- Branching narrative with multiple endings
- Original 16x16 and 32x32 pixel art created in Aseprite
- Crafting and equipment upgrade system
- Original chiptune soundtrack
- Gamepad and keyboard support

## Tech Stack

- **Engine:** Godot 4.2
- **Language:** GDScript + custom shaders
- **Art:** Aseprite for sprites and animations
- **Audio:** FMOD for adaptive audio
- **Platforms:** PC (Steam), console-exportable
