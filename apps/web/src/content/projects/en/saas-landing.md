High-performance landing page for an educational technology startup. Designed to maximize conversion with A/B testing, smooth animations, and perfect Lighthouse scores across all categories. Generates leads and registrations for the SaaS platform.

## Key Features

- Lighthouse score 100/100 in Performance, SEO, A11y, and Best Practices
- Scroll animations with Framer Motion and CSS
- Sections: hero with video, features, pricing, testimonials, FAQ, CTA
- Registration form with client-side validation
- Integrated A/B testing with Posthog
- Blog integrated with MDX for content marketing
- i18n (English and Spanish)

## Tech Stack

- **Framework:** Astro 5 with islands architecture
- **Styles:** Tailwind CSS 4 with custom theme
- **Animations:** Framer Motion + CSS animations
- **Analytics:** Posthog for A/B testing and funnels
- **Forms:** Resend for transactional emails
- **Deploy:** Vercel with preview deploys per PR
- **CDN:** Vercel Edge + Cloudflare Images
