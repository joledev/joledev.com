Landing page de alto rendimiento para startup de tecnología educativa. Diseñada para maximizar conversión con A/B testing, animaciones fluidas y Lighthouse score de 100 en todas las categorías. Genera leads y registros para la plataforma SaaS.

## Funcionalidades principales

- Score Lighthouse 100/100 en Performance, SEO, A11y y Best Practices
- Animaciones de scroll con Framer Motion y CSS
- Secciones: hero con video, features, pricing, testimonials, FAQ, CTA
- Formulario de registro con validación client-side
- A/B testing integrado con Posthog
- Blog integrado con MDX para content marketing
- i18n (español e inglés)

## Stack técnico

- **Framework:** Astro 5 con islands architecture
- **Estilos:** Tailwind CSS 4 con theme custom
- **Animaciones:** Framer Motion + CSS animations
- **Analytics:** Posthog para A/B testing y funnels
- **Forms:** Resend para emails transaccionales
- **Deploy:** Vercel con preview deploys por PR
- **CDN:** Vercel Edge + Cloudflare Images
